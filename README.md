# 웹 도우미

웹페이지 번역, 텍스트 번역, 검색 보조, SEO 진단, 반복 체크리스트를 한 사이드패널에서 처리하는 Chrome 확장 프로그램입니다.

개발 문서: [AGENTS.md](./AGENTS.md)

## 현재 버전
- 버전: `2.4.0`
- 마지막 정리: `2026-03-12`

## 주요 기능
- `초고속 번역`
  - API Key 없이 현재 페이지를 빠르게 읽을 수 있게 번역합니다.
- `AI 정밀 번역`
  - 현재 선택한 provider/model로 제목, 인용문, 경고문, 고유명사 보존을 더 우선해 번역합니다.
- `번역 기록`
  - 번역 탭 안에서 최근 번역 기록을 확인하고 다시 실행할 수 있습니다.
- `선택 텍스트 번역`
  - 드래그 후 플로팅 버튼으로 빠르게 번역하고 복사할 수 있습니다.
- `텍스트 번역`
  - 짧은 텍스트를 직접 입력해 번역할 수 있습니다.
- `스마트 검색`
  - 검색 의도 기반 추천 검색문으로 여러 검색 엔진을 열 수 있습니다.
- `SEO 검사`
  - 검색 노출과 AI 노출 요소를 함께 점검합니다.
- `반복관리`
  - 카테고리형 반복 체크리스트를 관리할 수 있습니다.

AI 정밀 번역은 설정에서 provider API Key를 입력해야 사용할 수 있습니다.

## 설치 방법
1. 이 저장소를 내려받거나 clone 합니다.
2. Chrome에서 `chrome://extensions/`를 엽니다.
3. 우측 상단 `개발자 모드`를 켭니다.
4. `압축해제된 확장 프로그램 로드`를 눌러 이 폴더를 선택합니다.
5. 툴바 또는 단축키 `Ctrl+Shift+Y`로 사이드패널을 엽니다.

## 기본 사용 흐름
1. `설정` 탭에서 필요한 provider API Key를 입력합니다.
2. `번역` 탭에서 provider와 모델을 고릅니다.
3. `초고속 번역` 또는 `AI 정밀 번역`을 실행합니다.
4. 필요하면 같은 탭의 `번역 기록` 서브탭에서 최근 결과를 다시 엽니다.
5. 페이지에서 텍스트를 드래그하면 선택 번역 버튼으로 즉시 번역할 수 있습니다.

## 업데이트 방법
1. 최신 코드를 내려받아 기존 폴더에 덮어씁니다.
2. `chrome://extensions/`에서 `웹 도우미`를 다시 로드합니다.
3. 이미 열려 있던 웹페이지 탭도 새로고침합니다.
   - content script가 업데이트되려면 열린 탭 새로고침이 필요할 수 있습니다.

## 릴리스 체크
- `manifest.json` 버전 확인
- `meta.js` 날짜 확인
- 변경한 JS 파일 `node --check`
- 필요 시 `node scripts/test-provider-models.mjs`
- 확장 다시 로드 후 사용 탭 새로고침
