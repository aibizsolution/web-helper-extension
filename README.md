# 웹 도우미

웹페이지 번역, 텍스트 번역, 검색 보조, 페이지 진단, 브라우저 도구, 반복 체크리스트를 한 사이드패널에서 처리하는 Chrome 확장 프로그램입니다.

개발 문서: [AGENTS.md](./AGENTS.md)  
변경 기록: [CHANGELOG.md](./CHANGELOG.md)

## 현재 버전
- 버전: `2.4.0`
- 마지막 정리: `2026-03-13`

## 주요 기능
- `번역`
  - 메인 탭 하나 안에서 `사이트 / 텍스트 / 기록` 서브탭으로 번역 작업을 묶어 제공합니다.
- `사이트 번역`
  - API Key 없이 빠르게 읽는 초고속 번역과, 선택한 provider/model을 쓰는 AI 정밀 번역을 제공합니다.
- `텍스트 번역`
  - 짧은 텍스트를 직접 입력해 번역하고, 같은 서브탭에서 기록도 다시 볼 수 있습니다.
- `선택 텍스트 번역`
  - 드래그 액션바와 우클릭 메뉴에서 번역/복사/설명하기를 빠르게 실행할 수 있습니다.
- `스마트 검색`
  - 검색 의도 기반 추천 검색문으로 여러 검색 엔진을 열 수 있습니다.
- `페이지 진단`
  - 검색 노출과 AI 노출 요소를 함께 점검합니다.
- `도구`
  - 브라우저 캐시/사이트 데이터 정리와 기간별 방문 TOP 확인을 제공합니다.
- `오류 센터`
  - 최근 오류/경고 로그 확인과 요약 복사, 로그 정리를 제공합니다.
- `반복관리`
  - 카테고리형 반복 체크리스트를 관리할 수 있습니다.

## UI 구조 원칙
- `sidepanel.html`은 패널 골격과 접근성용 마크업 위주로 유지합니다.
- 스타일은 `styles/` 아래 `tokens`, `layout`, `components`, `feature-panels`로 분리합니다.
- 메인 탭 lifecycle은 `modules/tab-registry.js`와 `modules/panel-tab-modules.js`를 통해 연결합니다.
- 기준은 `모바일 대응`이 아니라 `데스크톱 Chrome 사이드패널 폭 최적화`입니다.

## 설치 방법
1. 이 저장소를 내려받거나 clone 합니다.
2. Chrome에서 `chrome://extensions/`를 엽니다.
3. 우측 상단 `개발자 모드`를 켭니다.
4. `압축해제된 확장 프로그램 로드`를 눌러 이 폴더를 선택합니다.
5. 툴바 또는 단축키 `Ctrl+Shift+Y`로 사이드패널을 엽니다.

## 처음 받는 사람용 빠른 안내
- 이 확장은 `압축해제된 확장 프로그램 로드`로 바로 실행할 수 있습니다.
- 현재 저장소에는 `package.json`이 없어서 `npm install` 기반 프로젝트는 아닙니다.
- 즉, `사용`만 목적이면 `npm install` 없이 clone 후 바로 Chrome에 로드하면 됩니다.
- `Node.js`는 확장을 쓰는 데 필수는 아니고, `node --check`, `scripts/test-provider-models.mjs` 같은 검증 작업에만 필요합니다.
- `.env.providers.local`은 로컬 전용 파일이며 커밋하지 않습니다. provider smoke test가 필요할 때만 직접 만들어 사용합니다.

## 개발/검증 전제
- 현재 저장소는 브라우저 확장 실행 자체에는 별도 빌드가 필요 없습니다.
- 다만 개발 보조 도구를 `npm install` 한 번으로 복원하는 구조는 아직 아닙니다.
- 그래서 새 개발자가 받아서 바로 할 수 있는 최소 경로는 `clone -> 압축해제된 확장 프로그램 로드 -> 필요 시 Node로 문법 체크`입니다.
- 향후 온보딩을 더 단순하게 하려면 `package.json`과 스크립트 정리가 추가로 필요합니다.

## 기본 사용 흐름
1. `설정` 탭에서 필요한 provider API Key를 입력합니다.
2. `번역` 탭의 `사이트` 서브탭에서 provider와 모델을 고릅니다.
3. `초고속 번역` 또는 `AI 정밀 번역`을 실행합니다.
4. 텍스트를 직접 번역하려면 같은 탭의 `텍스트` 서브탭을 사용합니다.
5. 필요하면 같은 탭의 `기록` 서브탭에서 최근 결과를 다시 엽니다.
6. 페이지에서 텍스트를 드래그하면 액션바로 번역/복사/설명하기를 빠르게 실행할 수 있습니다.

## 업데이트 방법
1. 최신 코드를 내려받아 기존 폴더에 덮어씁니다.
2. `chrome://extensions/`에서 `웹 도우미`를 다시 로드합니다.
3. 이미 열려 있던 웹페이지 탭도 새로고침합니다.
   - content script가 업데이트되려면 열린 탭 새로고침이 필요할 수 있습니다.

## 회사/집에서 번갈아 작업할 때
- 가능합니다. 각 PC에서 같은 Git 저장소를 clone 받아 쓰면 됩니다.
- 각 PC/브라우저 프로필마다 한 번씩 `압축해제된 확장 프로그램 로드`를 해줘야 합니다.
- 코드 변경을 다른 PC에서 이어서 하려면 `git push -> 다른 PC에서 git pull -> chrome://extensions 에서 다시 로드 -> 열려 있던 탭 새로고침` 순서로 보면 됩니다.
- `설정`, `API Key`, `드래그 액션바 on/off`, 기본 provider/model 같은 값은 `chrome.storage.local`에 저장되므로 GitHub로 같이 이동하지 않습니다.
- 즉 회사 PC와 집 PC 사이에서 확장 설정값은 자동 동기화되지 않으며, 각 PC에서 다시 입력하거나 맞춰줘야 합니다.
- `.env.providers.local`도 로컬 전용이라 각 PC에 따로 준비해야 합니다.
