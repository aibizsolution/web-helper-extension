# 변경 기록

## 2026-03-13
- 우측 탭바를 `번역 / 페이지 진단 / 검색 / 도구 / 반복관리 / 설정 / 오류 센터 / 최신버전` 구조로 재정리했다.
- `사이트`와 `텍스트`를 메인 탭에서 분리하지 않고, `번역` 탭 안의 `사이트 / 텍스트 / 기록` 서브탭으로 통합했다.
- 사용자 노출 문구의 `초고속 번역` 계열 이름을 `구글 번역`으로 정리했다.
- 선택 텍스트 드래그 액션바와 우클릭 메뉴에 검색 엔진 바로 열기 기능을 추가했다.
- `package.json`, `npm run check`, `npm run test:providers` 기준으로 개발 온보딩을 단순화했다.
- 우측 탭바의 `페이지 진단` 버튼은 사이트 모양 아이콘과 `진단` 라벨 조합으로 더 컴팩트하게 조정했다.
- `최신버전` 버튼과 provider 테스트 스크립트의 GitHub 저장소 링크를 `aibizsolution/web-helper-extension` 공개 저장소 기준으로 맞췄다.
- `오류 센터` 탭을 추가해 최근 오류/경고 로그 확인, 요약 복사, 로그 삭제를 지원한다.
- 업데이트 일자와 변경 문서 관리를 위한 운영 규칙을 AGENTS에 추가했다.

## 2026-03-12 · v2.4.0
- 제품명을 `웹 도우미`로 정리했다.
- 번역 탭을 `현재 페이지 / 번역 기록` 구조로 재편했다.
- 초고속 번역과 AI 정밀 번역 UX를 분리했다.
- provider/model 선택을 멀티 provider 구조로 정비했다.
- 선택 텍스트 번역 UX와 복사 흐름을 개선했다.
- 사이드패널 단축키 토글을 추가했다.
- 개발 문서를 `AGENTS.md` 기준으로 재정비했다.
